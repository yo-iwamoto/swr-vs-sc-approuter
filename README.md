# yo-iwamoto/swr-vs-sc-approuter

SWR vs Server Components for server state management on App Router.

## Status

- [ ] SWR
  - wip
- [ ] Server Components

## Getting Started

```shell
pnpm i

pnpm db:push

pnpm db:seed

pnpm dev
```

Now you can open [http://localhost:3000](http://localhost:3000) in your browser.

## Unit test

```shell
pnpm run test
```
